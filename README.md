# Football Trivia Battle

A pixel-art football trivia game. Answer trivia to take and save penalties in a
1 v CPU shootout. Questions come from the Open Trivia Database.

See [Plans/Initial Plan.md](Plans/Initial%20Plan.md) for the Phase 1 plan.

## Stack

- Vite + React + TypeScript

## Scripts

```bash
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # typecheck + production build
npm run typecheck# type-check only
npm run lint     # oxlint
```
